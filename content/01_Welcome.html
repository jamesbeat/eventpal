Predefined base styles for all HTML elements, based on the Bourbon SASS library.

Webfonts

Webfonts via Googlefonts. Per Default, Montserrat and Lato are anabled. This can be changed in sass/variables.scss

Responsive Grid

Fully responsive Grid based on Bourbon NEAT. Default breakpoints are: 1088px, 768px and 480px. These can be defined in sass/config.scss
